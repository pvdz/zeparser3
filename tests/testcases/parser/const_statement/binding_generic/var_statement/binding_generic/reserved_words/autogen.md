### Templates

#### var statement

`````js
var # = x;
`````

#### for header

`````js
for (var # = x;;);
`````

#### for-in header

`````js
for (var # in y);
`````

#### for-of header

`````js
for (var # of y);
`````

#### function name

`````js
function #() {}
`````

#### function arg

`````js
function f(#) {}
`````

#### function object destructured arg

`````js
function f({#}) {}
`````

#### function object alias destructured arg

certain tokens fail because they are part of a multi-token expression

`````js
function fh({x: #}) {}
`````

#### function array destructured arg

`````js
function f([#]) {}
`````

#### catch clause

`````js
try {} catch (#) {}
`````

#### export

`````js

`````

#### can be property
